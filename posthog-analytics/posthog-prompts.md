# PostHog Max Prompts

## 1. Growth Accounting (Lifecycle)

Create a Lifecycle insight tracking the $pageview event with a weekly interval over the last 90 days. This should show new, returning, resurrecting, and dormant users. Save it as "TRMNL - Growth Accounting".

## 2. Weekly Active Users

Create a Trends insight showing unique users who performed the $pageview event, aggregated weekly over the last 90 days. Display it as a line chart. Save it as "TRMNL - Weekly Active Users".

## 3. Loan Application Events

Create a Trends insight showing the total count of loan_application_created events per day over the last 30 days. Display it as a line chart. Save it as "TRMNL - Loan Applications".
